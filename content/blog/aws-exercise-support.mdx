### ENDPOINTS:

#### Start Lab / Create AWS Account (Platform)

**Method**: `POST` <br />
**Endpoint**: `/aws/:exerciseId` <br />
**Response:**

```jsx
credentials: {
  username: string;
  password: string;
  consoleUrl: string;
}
```

#### Check Exercise / Check Test Case URLs (compare actual & expected outputs)

**Method**: `POST` <br />
**Endpoint**: `/aws/:exerciseId/check` <br />
**Response:**

```jsx
[
	{
		actualOutput: string
		errorMessage: string
		input: string
		output: string
		result: string
		testCaseName: string
		stdoutDiff: [
			{
				count: number
				value: string
			}
			...
		]
	}
	...
]
```

  src="https://sherbolotarbaev.co/images/aws-exercise-support2.png"

### Get AWS Account credentials (Platform)

**Method**: `GET` <br />
**Endpoint**: `/aws/:exerciseId` <br />
**Response:**

```jsx
{
  username: string;
  password: string;
  consoleUrl: string;
  accountId: number;
}
```
